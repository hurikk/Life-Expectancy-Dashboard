Project Name: Data Dashboard Project

Target: This project was created with the intention of testing my Python skills in an ETL process where the results of post-ETL processing are used in an interactive dashboard.

Dataset Name: Our World in Data (OWID)
About The Dataset: The dataset used is based on the average life expectancy of people from 1950 to 2023 in various countries.
Data Source: https://ourworldindata.org/grapher/life-expectancy.csv?v=1&csvType=full&useColumnShortNames=true
